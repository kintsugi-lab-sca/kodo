## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-03-01 | Information Disclosure | HOME override scope | mitigate | `process.env.HOME = tmpHome` antes del dynamic import de `state.js` (CR-02). El child del spawnSync recibe `env: { HOME: tmpHome }` explícito. Doble defensa: runner y child apuntan al mismo tmpdir aislado. |

| T-17-03-03 | Denial of Service | spawnSync timeout | mitigate | `spawnSync(..., { timeout: 5000 })` — el CLI hard-cap. El proceso se mata con SIGTERM si excede. Asserts de exit code cubren tanto exit normal como timeout (status null = timeout). |
