## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| tarball de release → `brew install` | fuente externa descargada e instalada en la máquina del usuario |
| launchd (plist) → proceso daemon | supervisor del OS arranca el binario con PATH mínimo |
| plist (world-readable) → secretos | `~/Library/LaunchAgents/*.plist` es legible por cualquier usuario local |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-66-08 | Information Disclosure | plist environment_variables | high | mitigate | OMITIR `environment_variables`; secretos en ~/.kodo/.env (0600), cargados en runtime por config.js (boundary PERSIST-04) |
