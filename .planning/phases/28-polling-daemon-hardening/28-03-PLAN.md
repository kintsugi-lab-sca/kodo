## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Padre `runPollingStartCli` → `openSync` → kernel fd | Path bajo `~/.kodo/logs/` (HOME-isolated). Mode 0o600 evita lectura por otros usuarios. |
| Child stdout/stderr → fd inherited via `spawn detached` | Kernel propaga fd al hijo via dup; child no abre el archivo directamente. |

| Daemon crash → stdout/stderr → logfile | Texto crudo del proceso. Puede contener stack traces (paths absolutos del sistema), pero no secrets del entorno (el fd captura stderr de Node, no env vars). |

| T-28-10 | S (Spoofing) | Symlink attack en `~/.kodo/logs/polling-YYYY-MM-DD.log` | mitigate | `openSync(path, 'a', 0o600)` con flag `a` (append) abre el target del symlink si existe.
