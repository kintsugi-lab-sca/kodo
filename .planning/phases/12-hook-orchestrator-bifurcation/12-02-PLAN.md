**Cuerpo nuevo (reemplaza líneas 39-46):**

```javascript
export function buildStopNudgeText(session) {
  const base = `La sesión ${session.task_ref} (${session.summary}) ha terminado y está en Review.`;
  switch (getSessionMode(session)) {
    case 'quick':
      // D-08: texto ES, NO sugiere verify. Escape literal `\\n` preservado (D-04 Phase 10).
      return `${base} Es una sesión GSD quick (one-shot, sin VERIFICATION.md). Revísala manualmente como cualquier sesión no-GSD.\\n`;
    case 'full': {
      // Texto Phase 10 D-04 preservado verbatim.
      const phaseLabel = session.phase_id ? `fase ${session.phase_id}` : 'bootstrap';
      return `${base} Es una sesión GSD (${phaseLabel}). Ejecuta \`kodo gsd verify ${session.session_id}\` y actúa según el verdict.\\n`;
    }
    default:
      // null → sesión no-GSD. Texto original preservado.
      return `${base} Revisa el resultado y decide si pasa a Done o necesita más trabajo.\\n`;
  }
}
```

**Validaciones críticas:**

1. **Escape literal `\\n`**: en los 3 returns el final es `.\\n\`` (dos chars en source: `\` + `n`). NO cambiar a `\n` (newline real). `cmux.send` interpreta `\\n` como Enter; un newline real no se transmite igual. Patrón establecido en stop.js Phase 10 y replicado en `launch.js:54,90`.

2. **Bloque case 'full' con braces `{ }`**: necesario porque declara `const phaseLabel`. Sin braces, `const` haría hoisting al case siguiente y rompería al lint. Los otros dos cases NO necesitan braces (return directo).

3. **D-09 KISS**: el case quick NO sugiere acciones adicionales (convertir a fase, abrir nueva task, etc.). Una sola frase ES.

4. **D-10**: NO tocar el bloque `if (session.gsd) { releaseGsdLock(...) }` que vive ~80 líneas más abajo (alrededor de línea 127). `session.gsd === true` para ambos modos por D-04 Phase 11, así que el lock release sigue cubriendo quick correctamente.

Por qué se hace así:
- D-07: switch exhaustivo simétrico con `dispatcher.js:153`. Phase 13 testeará los 4 estados (legacy, full+phase, full+bootstrap, quick, no-GSD) sobre este patrón.
- El helper aplica D-08 Phase 11 (legacy `gsd:true` sin `gsd_mode` → `'full'`). El switch hereda esta semántica sin replicar la regla.
- D-09: KISS — quick es deliberadamente liviano.
