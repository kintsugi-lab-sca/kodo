## Implementation Details

### Edit 1 — `KODO_ROOT` con env override (L20)
```javascript
// antes
const KODO_ROOT = join(__dirname, '..', '..');
// después
const KODO_ROOT = process.env.KODO_ROOT || join(__dirname, '..', '..');
```
Aditivo: `process.env.KODO_ROOT` no está set en producción, fallback al cálculo derivado. Análogo a la convención `KODO_DIR` vía `homedir()` en `src/session/state.js`.

### Edit 2 — `SKILL_PATH` (L21)
