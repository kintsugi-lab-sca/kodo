# Milestone v0.6: Session Isolation & Skill Sync

**Status:** ✅ SHIPPED 2026-05-13
**Phases:** 18-22
**Total Plans:** 12

## Overview

Milestone v0.6 aísla las sesiones de trabajo kodo en git worktrees por defecto, sincroniza la canonical skill `kodo-orchestrate` automáticamente entre repo y home, inyecta un recordatorio anti-push-fantasma a TODAS las sesiones (GSD full + quick + no-GSD), y cierra el Resolution Log acumulado en v0.5 (Phases 14/15/16). Resuelve la incidencia 28/04 (ROMAN-113…118) donde `git add -A` cross-staged entre sesiones paralelas, y bloquea el patrón ROMAN-125/126 donde el agente parafraseaba "deploy hecho" sin pushear.

## Milestone Summary

**Key Decisions:**

- Worktree always-on en lugar de label opt-in (Phase 18 D-01) — resuelve la incidencia ROMAN-113…118 por construcción.
- Lock per-repo invariante, NUNCA sobre worktree (Phase 18 D-03 → SC#3) — preserva la coalescencia Phase 8 GSD-10.
- Orchestrator EXCLUIDO de `--worktree` (Phase 18 D-06) — necesita cwd=repo para auto-cargar la skill local Phase 999.1.
- Cleanup fail-open con `<wt>.dirty` move-aside (Phase 19 D-02/D-03) — prioriza preservar trabajo no committed sobre repo limpio.
- `markSessionStatus` universal post-CR-02 (Phase 19 CR-02) — TODAS las sesiones (GSD + no-GSD) transitan a `'done'` PRE-sessionEnd.
- HOOK-01 idioma split preservado (Phase 20 D-01) — ES en `buildSessionContext`, EN en `buildGsdContext` (Phase 8 D-04 invariante).
- HOOK-01 append AL FINAL (Phase 20 D-03) — golden bytes preserved by construction; tags GSD intactos.
- Skill sync SHA-256 hash (Phase 21 D-02) — robusto contra `touch` / git checkout; sin nuevas deps (`node:crypto`).
- Auto-sync fail-open con event NDJSON (Phase 21 D-03) — no rompe el launch del orchestrator si home queda stale.
- Symlink legacy detect + replace (Phase 21 D-04) — resuelve el residuo Phase 999.1 (`~/.claude/skills/kodo-orchestrate` → path inexistente) en el primer ejecución.
- ANSI_*/COLOR_BY_LEVEL retirados; ANSI_RESET + ANSI_RED siguen privadas (Phase 22 D-05a..D-05e) — writeNdjson error path intacto.
