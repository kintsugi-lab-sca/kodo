## Copywriting Contract

This is a read-only dashboard cell, not an interactive control — there is no CTA and no
destructive confirmation in scope. The "copy" that matters is the exact rendered output for each
state of the `next` cell.

| Element | Copy |
|---------|------|
| Primary CTA | none (read-only dashboard render; no button/action added this phase) |
| Empty state (missing/cleared `next`) | `''` — empty cell, **no placeholder glyph** (SC5, LOCKED). A cleared `next` from DEBT-01 renders here, byte-identical to a never-set `next`. |
| Populated state | The persisted `next` string **verbatim**, with internal whitespace runs collapsed to a single space and outer whitespace trimmed. |
| Error / malformed input | `''` — non-string `session.next` returns empty; `nextCell` **never throws** (D-07, dashboard never-throws invariant). |
| Destructive confirmation | not applicable (no destructive action in this phase) |

Note on empty vs. placeholder: unlike `progCell` (which uses `—` for the no-progress state),
`nextCell` deliberately renders `''` for empty — this asymmetry is intentional and LOCKED (SC5,
"degradación limpia, sin placeholder ruidoso"). Do NOT introduce a `—` or any placeholder into
`nextCell`.

## Render Contract — `nextCell` (DEBT-03)

The exact, prescriptive transformation the executor implements at `src/cli/dashboard/format.js:258`.

**Input:** `session.next` (`string | null | undefined`, or any non-string via a hand-edited
`state.json`). Persisted **verbatim** upstream — the datum is NOT sanitized on write nor on the
App.js enrich beyond the existing `stripControlChars` (D-06; Phase 78 lane intact).

**Transformation (D-05):**
1. If `session.next` is not a string → return `''`.
2. Collapse every whitespace run with `/\s+/g` → single space (covers `\n`, `\t`, `\r`, multiple spaces).
3. `trim()` the result.
4. If the result is empty → return `''` (empty cell, no placeholder — SC5).
5. Otherwise return the collapsed, trimmed string.

**Invariants the transformation must preserve:**
- **Pure & never-throws (D-07):** no side effects, no color, non-string → `''`.
- **Verbatim persistence (D-06):** the collapse is render-only; `state.tasks[taskId].next` is
  untouched. Do NOT move the collapse into `upsertTaskHandoff` or the App.js enrich.
- **Layered sanitization (Phase 78):** `stripControlChars` (enrich) handles control chars; this
  whitespace collapse is the complementary **layout** layer — not a replacement, not redundant.
- **SC5 empty cell:** whitespace-only input collapses to `''`, never a placeholder.

Reference shape (executor finalizes wording/style):
```js
export function nextCell(session) {
  return typeof session.next === 'string'
    ? session.next.replace(/\s+/g, ' ').trim()
    : '';
}
```
