| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-66-05 | Information Disclosure | `status --json` | low | mitigate | La salida emite SOLO {status, pid}; jamás config, env, secretos ni rutas (boundary PERSIST-04 intacto) |
| T-66-06 | Repudiation | exit codes no deterministas | low | accept | status siempre exit 0 por diseño (D-13); scripts distinguen estado por el JSON, no por el code |
</threat_model>
